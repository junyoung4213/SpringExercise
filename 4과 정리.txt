<STS (SpringSource Tool Suite) 소개>
: Spring 개발업체인 SpringSource가 직접 만들어 제공하는 이클립스의 확장판으로
최신 이클립스를 기반으로 주요한 Spring 지원 플러그인과 관련된 도구를 모아서 Spring 개발에 
최적화 되도록 만들어진 IDE 이다.

<Maven이란?>
: 라이브러리 관리 + 빌드 툴이다.

<사용하는 이유?>

1. 편리한 Dependent Library 관리 - Dependency Management

2. 여러 프로젝트에서 프로젝트 정보나 jar파일들을 공유하기 쉬움

3. 모든 프로젝트의 빌드 프로세스를 일관되게 가져갈 수 있음.
